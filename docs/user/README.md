# 用户手册

## 📖 概述

欢迎使用 Ting Learning Platform！这是一套基于视频音频学习赚积分的综合性学习平台，通过学习内容获取积分，积分可用于兑换商品或服务。

## 🚀 快速入门

### 1. 注册登录

#### 1.1 微信登录 (推荐)
1. 打开小程序或访问网站
2. 点击"微信登录"按钮
3. 授权微信登录
4. 完善手机号绑定（首次登录）

#### 1.2 账号注册
1. 点击"注册"按钮
2. 输入用户名、手机号
3. 设置密码（至少6位）
4. 完成注册

#### 1.3 账号登录
1. 点击"登录"按钮
2. 输入用户名和密码
3. 登录成功

### 2. 开始学习

#### 2.1 浏览内容
- 在首页或分类页面浏览视频/音频内容
- 点击感兴趣的内容开始学习
- 查看内容详情、时长、积分奖励

#### 2.2 观看/收听
- 点击播放按钮开始学习
- 支持横竖屏切换（视频）
- 支持倍速播放
- 支持断点续播
- 完成学习后自动获得积分

#### 2.3 完成答题
- 学习结束后可能需要答题
- 答题正确可获得额外积分
- 答题错误可重新尝试（有限制）

### 3. 积分管理

#### 3.1 查看积分
- 进入"我的积分"页面
- 查看总积分、可用积分
- 查看积分获取记录
- 查看积分消耗记录

#### 3.2 积分获取
- **观看视频**: 按时长获得积分
- **收听音频**: 按时长获得积分
- **答题奖励**: 答题正确获得积分
- **每日签到**: 连续签到获得积分
- **邀请好友**: 成功邀请获得积分

#### 3.3 积分有效期
- 积分可能有过期时间
- 关注"即将过期"提醒
- 及时使用即将过期的积分

### 4. 积分兑换

#### 4.1 浏览商品
- 进入"积分商城"页面
- 浏览可兑换的商品
- 查看商品详情和所需积分

#### 4.2 兌换商品
- 选择想要的商品
- 点击"兑换"按钮
- 确认兑换信息
- 扣除相应积分
- 生成兑换码

#### 4.3 线下核销
- 凭兑换码到指定门店
- 向店员出示兑换码
- 完成核销

## 👥 邀请推广

### 2. 分享邀请

### 3. 查看收益

- 查看下级用户学习情况

## 🆘 常见问题

### Q1: 如何获得更多积分？
A: 通过以下方式可以获得积分：
- 观看/收听学习内容
- 正确回答课后问题
- 每日签到
- 邀请新用户注册
- 参与平台活动

### Q2: 积分可以兑换什么？
A: 积分可以兑换：
- 虚拟商品（优惠券、会员等）
- 实体商品（实物礼品）
- 服务（课程、咨询等）
- 其他特权

### Q3: 积分会过期吗？
A: 是的，部分积分可能有有效期，建议及时使用。

### Q4: 邀请好友如何获得奖励？
A: 成功邀请好友注册并完成首次学习后，您将获得相应积分奖励。

### Q5: 兌换的商品如何核销？
A: 1. 在积分商城兑换商品
2. 获得兑换码
3. 前往指定门店
4. 向店员出示兑换码
5. 完成核销
